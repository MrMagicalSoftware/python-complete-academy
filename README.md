**data types**


Python has the following data types built-in by default, in these categories:
Text Type: 	str
Numeric Types: 	int, float, complex
Sequence Types: 	list, tuple, range
Mapping Type: 	dict
Set Types: 	set, frozenset
Boolean Type: 	bool
Binary Types: 	bytes, bytearray, memoryview
None Type: 	NoneType



1. **Integer (int):**
   ```python
   age = 25
   ```

2. **Float (float):**
   ```python
   temperature = 98.6
   ```

3. **String (str):**
   ```python
   name = "John Doe"
   ```

4. **Boolean (bool):**
   ```python
   is_student = True
   ```

5. **List:**
   ```python
   numbers = [1, 2, 3, 4, 5]
   ```

6. **Tuple:**
   ```python
   coordinates = (3.5, 7.2)
   ```

7. **Set:**
   ```python
   unique_numbers = {1, 2, 3, 4, 5}
   ```

8. **Dictionary:**
   ```python
   person = {'name': 'Alice', 'age': 30, 'city': 'New York'}
   ```

9. **NoneType (None):**
   ```python
   result = None
   ```
